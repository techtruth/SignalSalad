# Code of Conduct

If someone asked you to check this document,
you should think about why they asked that,
and it is advisable not to reply to them until tomorrow.

This "cute" suggestion suggest is currently sufficient.
If collective bad behavior forces this to advance,
I'm sure we can update this to reflect the needs.

## Enforcement

Project maintainers may remove or reject comments, issues, or contributions that
violate this policy.

Serious concerns can be reported via `SECURITY.md` contact channels.
